  <action>
Replace the current `renderBoard()` function with an incremental version that:

1. **On first call (empty board):** Creates all 9 cells as before, appending them. This is `boardEl.children.length === 0`.
2. **On subsequent calls:** Iterates over existing cell elements and only updates cells that have **changed state** — i.e., a cell that was empty is now X or O. For changed cells: update `textContent`, update `classList` (add `cell--taken`, `cell--x` or `cell--o`, remove `tabIndex`, update `aria-label`). For unchanged cells (already X or O): touch nothing — the CSS animation will NOT re-fire because the element was not recreated.
3. **Win highlighting:** When `winningPositions` is provided, add `cell--winning` to the winning cells. **Critically:** do NOT remove `cell--winning` from cells if they already have it (game-over rerender should be idempotent).
4. **Reset path:** `resetGame()` calls `boardEl.innerHTML = ''` BEFORE calling `renderBoard()` — this path is fine (total wipe on reset is intentional). Add `boardEl.innerHTML = ''` into `resetGame()` just before the `renderBoard()` call, OR detect the reset by checking if board is all-empty and `boardEl.children.length > 0`.

**Recommended implementation:**

```js
function renderBoard(winningPositions = []) {
  const board = game.get_board();  // Uint8Array[9]
  const winSet = new Set(winningPositions);

  // First render or after reset: build all cells from scratch
  if (boardEl.children.length !== 9) {
    boardEl.innerHTML = '';
    for (let i = 0; i < 9; i++) {
      const cell = document.createElement('div');
      cell.className = 'cell';
      cell.dataset.index = String(i);
      cell.setAttribute('role', 'gridcell');
      cell.tabIndex = 0;
      cell.setAttribute('role', 'button');
      cell.setAttribute('aria-label', `Empty cell ${i + 1}`);
      boardEl.appendChild(cell);
    }
  }

  // Incremental update: only patch cells whose state has changed
  const cells = boardEl.children;
  for (let i = 0; i < 9; i++) {
    const cell = cells[i];
    const value = board[i];
    const wasEmpty = !cell.classList.contains('cell--taken');

    if (value === PLAYER_X && wasEmpty) {
      cell.textContent = 'X';
      cell.classList.add('cell--taken', 'cell--x');
      cell.removeAttribute('tabindex');
      cell.setAttribute('role', 'gridcell');
      cell.setAttribute('aria-label', `X at position ${i + 1}`);
    } else if (value === PLAYER_O && wasEmpty) {
      cell.textContent = 'O';
      cell.classList.add('cell--taken', 'cell--o');
      cell.removeAttribute('tabindex');
      cell.setAttribute('role', 'gridcell');
      cell.setAttribute('aria-label', `O at position ${i + 1}`);
    }

    // Win highlight — applied once at game-over; idempotent
    if (winSet.has(i) && !cell.classList.contains('cell--winning')) {
      cell.classList.add('cell--winning');
    }
  }
}
```
