    ### ✅ THEM-01 — Light mode theme applies automatically

    **Chrome / Firefox / Safari — DevTools method:**
    1. Open DevTools → Rendering tab (Chrome: `...` menu → More Tools → Rendering; Firefox: `...` → Settings → Default theme → Light)
    2. Set "Emulate CSS prefers-color-scheme" to **light**
    3. **Expected:** Board background turns light grey (#f0f2f7), cells turn white, text turns dark navy — all without any page reload

    **macOS System Preferences method (alternative):**
    1. System Preferences → Appearance → Light
    2. Reload the page
    3. **Expected:** Same light theme as above

    ---

    ### ✅ THEM-02 — No FOUC (Flash of Unstyled Content)

    1. With light mode active (from above), do a hard reload (Cmd+Shift+R / Ctrl+Shift+R)
    2. Watch carefully during load
    3. **Expected:** Board appears in light theme immediately — no flash of dark theme before the CSS loads
    4. Switch back to dark mode, hard-reload again — board should appear dark immediately

    ---

    ### ✅ PERS-01 — Scores persist across page refresh

    1. Play 2–3 games until you have a non-zero score (e.g., 2 wins, 1 loss)
    2. Note the scoreboard values
    3. Reload the page (F5 / Cmd+R)
    4. **Expected:** Scoreboard shows the same values — wins, losses, and draws are preserved

    ---

    ### ✅ PERS-02 — Private/incognito mode degrades gracefully

    1. Open a **Private / Incognito** window (Cmd+Shift+N in Chrome, Cmd+Shift+P in Firefox)

    4. **Expected:** Game plays normally — no JavaScript errors in the console, score shows "0" values but does not crash
    5. Open DevTools Console — confirm: **no errors** related to localStorage, SecurityError, or storage
